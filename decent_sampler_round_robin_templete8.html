Decent Sampler Round Robin Template

Overview

This project provides a template for creating a Decent Sampler preset with round-robin sampling capabilities. The round-robin feature allows for alternating between different audio samples to create a more natural sound, commonly used to add variation in instruments like percussion, strings, or any repeated notes.

The preset uses XML format to define groups of samples, their playback behavior, and user interface controls, allowing musicians to easily tweak the settings. The template includes multiple user interface elements such as knobs for controlling LFO depth, attack, release, tone, and more.

Features

  • Round-Robin Playback: Alternates between groups of samples to create more variation in the sound.
  • User Interface Controls: Includes several labeled knobs to control key parameters such as attack, release, reverb, and tone.
  • Customizable UI: Background images, knobs, and other UI components can be easily customized.

XML Structure

Groups

The template defines multiple groups using the <groups> tag. These groups alternate based on the seqMode set to round_robin. This helps in creating a more realistic and dynamic playback.

<groups seqMode="round_robin" attack="0.062" decay="0.117" release="0.117">
    <group name="Group 1" seqPosition="1">
        <!-- YOUR SAMPLE GOES HERE -->
        <effects/>
    </group>
    <group name="Group 2" seqPosition="2">
        <!-- YOUR SAMPLE GOES HERE -->
        <effects/>
    </group>
</groups>

seqMode="round_robin": Ensures the groups are played in sequence, one after another.

Attack, Decay, Release: These attributes define the basic envelope for each group.

User Interface (<ui>)

The user interface elements are defined within the <ui> tag. It includes labeled knobs for adjusting parameters like LFO depth, attack, release, chorus, and tone.

<ui bgImage="Resources/background.png" width="812" height="375" layoutMode="relative" bgMode="top_left">
    <tab name="main">
        <labeled-knob x="355" y="15" width="120" label="LFO DEPTH" type="float" minValue="0.0" maxValue="0.32" value="0.0006">
            <binding level="instrument" type="modulator" position="0" parameter="MOD_AMOUNT"/>
        </labeled-knob>
        <!-- Additional knobs for ATTACK, RELEASE, etc. -->
    </tab>
    <keyboard/>
</ui>

<labeled-knob>: Defines a knob that users can interact with to adjust parameters.

Bindings: Connect the knobs to different instrument parameters, such as modulation or envelope settings.

How to Use

  1. Loading in Decent Sampler: Import the XML template into the Decent Sampler plugin.
  2. Add Samples: Replace the "YOUR SAMPLE GOES HERE" placeholders with your own audio samples.
  3. Customize UI: Update the knobs, labels, and background image to suit your instrument design.
  4. Adjust Parameters: Fine-tune attack, release, effects, and modulation to achieve the desired sound.

Example Use Case

This template is suitable for building an instrument with multiple articulations, such as a string instrument that uses round-robin to alternate between different bow strokes. You can use it to create realistic-sounding instruments with minimal effort by taking advantage of Decent Sampler's built-in capabilities.
